# The-Inner-Voice-VIS42-p5-final-project-
A story-driven game about parental expectations, self-doubt, and finding emotional validation through an unexpected companion.

## Genre:
- Interactive Fiction
- Emotional Companion
- ML-Sentiment Analysis
- Multimedia Integration

## Inspiration:
- Games: Gris, Celeste, Kind Words
- Academic: MIT Affective Computing, ELIZA
- Personal: Exploring themes of artistic identity and validation

## Libraries:
- p5.js (core framework)
- p5.sound (audio playback)
- ml5.js (machine learning): Sentiment Analysis
- WEBGL (3D rendering)

## Techniques
- 3D Room Construction
- Video Texture Mappin
- Real-time Video Capture
- DOM Element Integration
- State Management
- Scene Transitions
